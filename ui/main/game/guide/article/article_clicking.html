Clicking
[style=text_callout]Left Click[/style] to select unit, [style=text_callout]Right Click[/style] for action.
[style=text_callout]Shift + Right Click[/style] to queue up multiple orders
[style=text_callout]Left Click + Hold[/style] and drag to window select multiple units
[style=text_callout]Double click[/style] a unit to select similar units
